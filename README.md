# SIP Calculator (Powered by Marketyog)

A very attractive, advanced SIP calculator web app. Mobile, laptop, and tablet friendly. Features:

- SIP amount, duration, expected return rate
- Step-up SIP, inflation adjustment
- Investment summary
- Growth chart (Recharts)
- Downloadable PDF report
- Shareable results
- Dark/light mode
- PWA support (installable)
- Responsive design

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start development server:
   ```bash
   npm run dev
   ```
3. Open [http://localhost:5173](http://localhost:5173) in your browser.
